# Introduction

This is a training Full-Stack project, a chess game, to enhance my skills in pure Javascript, OOP, Node js, Express js, Mongo DB, and Web-Socket.

# Stack

* HTML&CSS <br>
* Javascript <br>
* OOP <br>
* Node js <br>
* Express js <br>
* PostgreSQL <br>
* Web-Socket <br>

# Setup

To run this project, install it locally using npm:

```
$ npm install
$ npm start
```
